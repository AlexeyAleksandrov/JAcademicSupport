Паспорт источника и формирование BPA
Параметры источника $s \in \mathcal{S} = \{\text{vac},\text{exp},\text{fc}\}$
  • Входной кортеж: $(n_s^{\text{tot}},\; n_s^{\text{rel}},\; \bar{a}_s,\; \alpha_s,\; w_s,\; \lambda_s)$
  • Эффективный объём: $n_s^{\text{eff}} = n_s^{\text{rel}} \cdot \alpha_s$
  • Коэффициент уверенности (Лаплас):
    $$\kappa_s = \frac{n_s^{\text{eff}}}{n_s^{\text{eff}} + \lambda_s} \;\in (0,1)$$
  • BPA (единая схема для всех источников):
    $m_s(\{T\}) = \kappa_s \cdot \bar{a}_s^{+} \cdot \dfrac{n_s^{+}}{n_s^{\text{tot}}}$ — масса «важно»
    $m_s(\{F\}) = \kappa_s \cdot \bar{a}_s^{-} \cdot \dfrac{n_s^{-}}{n_s^{\text{tot}}}$ — масса «не важно»
    $m_s(\Theta) = 1 - m_s(\{T\}) - m_s(\{F\})$ — неопределённость
  • Для vac, exp: нет отрицательных сигналов → $n_s^{-} = 0$ → $m_s(\{F\}) = 0$
  • Для fc (смешанный): $n_{\text{fc}}^{+} > 0,\; n_{\text{fc}}^{-} > 0$ → оба слагаемых ненулевые
Дисконтирование по надёжности $w_s$
$m'_s(A) = w_s \cdot m_s(A),\quad A \neq \Theta$
$m'_s(\Theta) = (1 - w_s) + w_s \cdot m_s(\Theta)$
Источник$w_s$$\lambda_s$

Рамка рассуждений
$\Theta = \{T,\,F\},\quad 2^\Theta = \bigl\{\emptyset,\{T\},\{F\},\Theta\bigr\}$
Выход: дисконтированные тройки $(m'_s(\{T\}),\; m'_s(\Theta),\; m'_s(\{F\}))$ для каждого источника $s$. Сумма равна 1. Передаются на последовательное комбинирование по правилу Демпстера.

Комбинирование и принятие решения
Пайплайн алгоритма
Конфликт и правило Демпстера
$$K = m_1(\{T\})\cdot m_2(\{F\}) + m_1(\{F\})\cdot m_2(\{T\})$$
$$(m_1 \oplus m_2)(A) = \frac{\displaystyle\sum_{B \cap C = A} m_1(B)\cdot m_2(C)}{1 - K},\quad K < \tau_K$$
$$m^* = m'_{\text{vac}} \oplus m'_{\text{exp}} \oplus m'_{\text{fc}}$$
Пигнистическая вероятность и дефицит
$$\text{BetP} = m^*(\{T\}) + \frac{m^*(\Theta)}{N_C},\qquad \Delta = \text{BetP} - \sigma$$
Правила принятия решений
$\Delta > \tau_\Delta \wedge K \le \tau_K \wedge m^*(\Theta) \le \tau_\Theta$ ⟹ УСИЛИТЬ
$\Delta > \tau_\Delta \wedge (K > \tau_K \vee m^*(\Theta) > \tau_\Theta)$ ⟹ ВАРИАТИВНОСТЬ
$|\Delta| \le \tau_\Delta$ ⟹ СТАБИЛИЗИРОВАТЬ
$\tau_\Delta = 0{,}15$;  $\tau_K = 0{,}40$;  $\tau_\Theta = 0{,}15$;  $N_C = 25$;  $\sigma$ — текущая доля кластера в РПД
Выход: рекомендация по кластеру компетенций — УСИЛИТЬ / ВАРИАТИВНОСТЬ / СТАБИЛИЗИРОВАТЬ / СОКРАТИТЬ. При $K \ge \tau_K$ — переключение на правило Ягера (конфликтная масса уходит в $\Theta$ вместо нормировки).
